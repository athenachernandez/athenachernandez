Computer science

Python

Python was the first programming language I learned. I took a software development course my freshman year of high school that was taught in Python. Our final project was to build an application of our choosing. I modeled my app over Google's Beethoven Game. Although it was challenging with the experience I had at the time, it was the first app I ever made and it inspired me to continue my path into the field of computer science.

Java

This is language I feel most comfortable with because I was able to learn it through 9 levels at The League of Amazing Programmers. You can learn more about my experience at the League by watching this video or copy-paste this link into a new tab: https://www.instagram.com/p/CD4bItCjoIR/. Currently, I'm on the last level preparing for the Java SE Oracle Certification. I built a COVID-19 app over the summmer of 2020; I was the project manager! You can clone the app repository from here or by copy-pasting this link into a new tab: https://github.com/LEAGUE-Level7/COVIDMAP. This app was made in Processing 3, which uses Java as its primary language. I'm also an intern, ambassador, and TA at the League because I feel confident in my programming skills and hope to share my knowledge with other young minds. As of now, I code in Java for programming competitions such as Codeforces.

C++

In the midst of learning the syntax for programming competitions because the run time is so fast!

Web Design

I've built a few websites from scratch, including this one. I'm familiar with HTML5, CSS, JavaScript, and Bootstrap 4. I created the website for a COVID application I developed; you can visit by clicking here or by copy-pasting https://level7.jointheleague.org/COVIDMAP into a new tab.

Figure skating

I'm a member of the La Jolla Figure Skating Club; I skate at UTC Ice. I used to compete frequently but since quarantine, I've decided to work towards my gold medals in MITF and free skating. I mostly only work on individual skating, but I've dabbled in pair skating.

Moves in the Field (MITF)

I'm currently working at the Junior level. I've passed prepreliminary, preliminary, prejuvenile, juvenile, intermediate, and novice moves.

Free skates

I'm currently working at the Juvenile level. I've passed prepreliminary, prelimiary, and prejuvenile free skates.

Music

I enjoy playing and listening to music. I've been playing the violin for 10 years. My instructor is Hernan Constantino. I enjoy playing in the school orchestra. Currently, I'm working on Mozart's Violin Concerto No. 5 in A Major K. 219. I also play the piano for fun; my goal is to be able to play Claire de Lune. I enjoy listening to all types of music. You can visit my Spotify here. I often play at retirement homes and hospitals with Melodies for Remedies — I'm an Outreach Manager there!
